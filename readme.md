### MPV 配置文件(win版) 
```
portable_config              # 目录放在主程序目录内，可作为配置文件根目
  script                       # 脚本目录
  script-opts                  # 脚本配置文件目录
  shaders                      # 视频增强配置文件
  watch_later                  # 文件播放进度缓存目录

mpv.con                      # 主配置文件
input.con                    # 快捷键、界面操作入口配置
  saved-props.json             # 自动保存的全局属性文件
```
