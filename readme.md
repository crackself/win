configuration files for windows program
`portable_config`            目录放在主程序目录内，可作为配置文件根目录
script                       为脚本目录
script-opts                  为脚本配置文件目录
shaders                      为视频增强配置文件
watch_later                  文件播放进度缓存目录
input.con                    快捷键、界面操作入口配置
mpv.con                      主配置文件
saved-props.json             自动保存的全局属性文件
